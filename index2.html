Como criar uma pasta e um arquivo HTML no VS Code 1. Criar uma pasta Abra o Visual Studio Code No menu superior, clique em File (Arquivo) → Open Folder (Abrir Pasta) Escolha ou crie uma nova pasta (ex: meu-site) Clique em Selecionar pasta Por que isso é importante? A pasta funciona como uma “caixa” onde você organiza todos os arquivos do seu projeto (HTML, CSS, imagens, etc). 2. Criar um arquivo HTML No lado esquerdo (Explorer), clique com o botão direito dentro da pasta Clique em New File (Novo Arquivo) Nomeie como: index.html Boa prática: Sempre usar .html no final index.html é o nome padrão que navegadores procuram primeiro 3. Escrever a estrutura básica Digite ! e pressione Enter (se tiver o Emmet ativado) Isso gera automaticamente a base do HTML. 4. Abrir no navegador Clique com o botão direito no arquivo Ou abra manualmente no navegador (duplo clique) Diferença entre HTML tradicional e HTML semântico HTML tradicional (não semântico) Exemplo:
